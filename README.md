# AdoteAqui

![adoteaqui](https://user-images.githubusercontent.com/93748083/144920525-f83aa9fb-0f37-4c1b-9bac-c47e53255aa8.png)

#### Aplicação online em: https://lesantxs.github.io/adoteaqui/

## Introdução

O projeto do curso Resilia Webdev, onde deveria ser feita uma página de adoção de cachorros e gatos.

## Tecnologias utilizadas

- HTML
- CSS

#### Objetivo

Desenvolver uma página de adoção de cães e gatos usando conhecimento em HTML, CSS, Flexbox para desenvolvimento de colunas e cards de animais.
